# Rogilla: Reproductor de Música

**Rogilla** es un reproductor de música desarollado usando el lenguaje Python y la interfaz Tkinter.
Fue el **primer trabajo práctico** realizado en *Laboratorio de Algoritmos*.

Tanto la decisión de *usar un animal como logo* (En este caso, un gorilla) como *la interfaz del reproductor*
están (ponele) *inspirados por el* ***Winamp clásico***.

Y no, el nombre no tiene nada que ver nada con *Gorillaz*, simplemente me gustaba como quedaba
(Además, suena más o menos parecido a ***Rolinga***, lo cuál fue accidental).

### Consideraciones

1. **Si no te gusta o le tienes fobia al color amarillo, no descargues Rogilla**.
Todo el *reproductor esta bañado de amarillo*, así que *no me hago cargo de cualquier daño ocular que te ocurra*.

2. Rogilla **solo puede reproducir archivos que tengas en tu computadora**,
así que si estabas buscando algo similar a Spotify, mejor busca en otro lado. 

3. Necesitaras **instalar éstas librerias en Python si quieres que Rogilla funcione**:

- **Pygame**: para reproducir la música.
- **TinyTag**: para obtener los metadatos (el titulo, artista y álbum) de los archivos.
- **Pillow**: para las portadas.
- **Fleep**: para filtrar los archivos de audio al añadirlos.

4. **Rogilla se encuentra (horriblemente) incompleto y (estupidamente) mal diseñado**
